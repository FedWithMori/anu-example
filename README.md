# anu-example
> 基于anujs + react-router 2.x + webpack 3.x + babel 6.x搭建的项目，由于官方给的IE8例子[anu-ie8-example](https://github.com/RubyLouvre/anu-ie8-example)太过于简单，基本的路由功能都不具备，所以自己折腾了一套

# 特性
1、基于anujs，支持react v16  
2、支持import()懒加载  
3、支持es5和es6  

# 使用说明：

1、安装依赖
```js
    npm i 
```

2、配置host
```js
    127.0.0.1 anu-example.com
```

3、运行项目
```js
    npm start
```
